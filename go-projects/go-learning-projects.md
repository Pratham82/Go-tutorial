# Go Learning Projects

A curated list of beginner-friendly projects to learn Golang, organized by difficulty.

---

## Beginner (Fundamentals)

1. **CLI Calculator** — Functions, arithmetic, user input
2. **Temperature Converter** — Celsius/Fahrenheit/Kelvin
3. **Number Guessing Game** — Random numbers, loops, conditionals
4. **Word Counter** — File reading, string manipulation

---

## Intermediate (Core Concepts)

5. **Todo CLI** — Slices, maps, structs, file I/O
6. **HTTP Server** — `net/http`, routing, handlers
7. **REST API (Book/Task Manager)** — JSON, CRUD operations
8. **Weather CLI** — HTTP requests, API integration, JSON parsing

---

## Advanced (Go Strengths)

9. **Chat Server** — WebSockets, goroutines, channels
10. **URL Shortener** — Database, REST API, hashing
11. **Concurrent File Downloader** — Goroutines, waitgroups
12. **JWT Auth Middleware** — Security, middleware patterns

---

## Projects by Concept

| Concept | Projects |
|---------|----------|
| **Basics** | Calculator, Converter, Guessing Game |
| **File I/O** | Word Counter, Notes App, File Organizer |
| **HTTP/JSON** | Weather CLI, REST API, URL Shortener |
| **Concurrency** | Chat Server, Downloader, Crawler |
| **Databases** | CRUD API, URL Shortener, Blog |

---

## Getting Started

1. Start with **Beginner** projects to grasp Go fundamentals
2. Move to **Intermediate** to learn Go's core packages and patterns
3. Tackle **Advanced** projects to master Go's concurrency model

Happy coding! 🚀
