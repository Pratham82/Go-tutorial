# CLI Calculator

A simple command-line calculator to learn Go fundamentals.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)

## Usage

```
Enter first number: 10
Enter operator (+, -, *, /, %): +
Enter second number: 5
Result: 15
```

## Learning Concepts

- `fmt` package (Print, Scan, Printf)
- Functions and parameters
- Type conversion (string → float64)
- Control flow with switch
- Error handling (invalid input)
